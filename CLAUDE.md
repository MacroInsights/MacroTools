# CLAUDE.md — MacroTools

## Project Overview

**MacroTools** is an R package (version 0.0.0.9000) that provides a collection of tools for macroeconomic analysis using United States data. It pulls data from the FRED (Federal Reserve Economic Data) and BLS (Bureau of Labor Statistics) APIs.

Author: Alfredo Romero (aaromero@ncat.edu)

## Project Structure

## Key Dependencies

- **fredr** — FRED API client
- **xts / tsibble / zoo** — time series data structures
- **dplyr / tidyr / purrr / magrittr** — data manipulation
- **leaflet / tmap** — interactive and static maps
- **memoise** — function result caching
- **httr / jsonlite / glue** — BLS API requests
- **lubridate** — date handling
- **gt / webshot2** — table rendering

## API Key Setup

Two API keys are required before using any data-fetching functions:

```r
enter_api_credentials(fred_api_key = "YOUR_FRED_KEY", bls_api_key = "YOUR_BLS_KEY")
```

Keys are stored in `~/.Renviron` and loaded into the global environment as `fredKey` and `blsKey`. **An R session restart is required after saving keys for the first time.**

## Development Notes

- Documentation is generated with **roxygen2** (`RoxygenNote: 7.3.2`). Do not edit `NAMESPACE` or `man/*.Rd` files by hand.
- Data-fetching functions use `memoise::memoise()` for in-memory caching to avoid redundant API calls.
- The package uses both the base pipe (`|>`) and the magrittr pipe (`%>%`) interchangeably.
